# Bharati Braille Specifications and Guidelines

Most Indian languages use Bharati Braille a system developed in the 1950's to unify as far as possible the Braille codes for the various Indian languages and scripts. The first PDF publication here outlines the latest version of Bharati Braille. It includes new rules which were introduced on 4th January 2025.
